Comparison and Logical Operators Explanation

Comparison Operators

Purpose: To compare two values and determine their relationship.

  • `==` (Equal to): Checks if two values are equal, ignoring their data types.
  • `===` (Strict equal to): Checks if two values are equal in both value and data type.
  • `!=` (Not equal to): Checks if two values are not equal.
  • `!==` (Strict not equal to): Checks if two values are not equal or are of different data types.
  • `>` (Greater than): Checks if the value on the left is greater than the value on the right.
  • `<` (Less than): Checks if the value on the left is less than the value on the right.
  • `>=` (Greater than or equal to): Checks if the value on the left is greater than or equal to the value on the right.
  • `<=` (Less than or equal to): Checks if the value on the left is less than or equal to the value on the right.

Logical Operators

Purpose: To combine multiple boolean expressions and return a boolean result based on the conditions.

  • `&&` (Logical AND): Returns true if both expressions are true.
  • `||` (Logical OR): Returns true if at least one of the expressions is true.
  • `!` (Logical NOT): Inverts the truth value of an expression; returns true if the expression is false, and false if the expression is true.
